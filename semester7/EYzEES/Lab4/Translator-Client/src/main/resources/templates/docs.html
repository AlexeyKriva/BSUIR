Обзор Использование API

Обзор

Этот проект предназначен для решения задач, связанных с автоматическим переводом текста с английского на немецкий и наоборот. Он использует API для перевода текста между различными языками. Также на выходе будет строится синтаксическое дерево, а пары слов будут сохранятся в словаре. Также API предоставляет возможность подсчёта слов в предложении и определения части речи слова.

Использование

После запуска сервера, вы можете посетить следующие страницы:

  • /home - Страница, на которой пользователь вводит текст для перевода, а также указывает язык текста и язык, на который пользователь хочет перевести исходный текст.
  • /vocabulary - Открытие страницы, позхволяющей пользователю получать все пары слов из словаря, добавлять в словарь новые пары, обновлять текущие пары, удалять текщуие пары слов, а также находить пары слов по id и по id запроса.
  • /documentation - Страница, на которой Вы находитесь сейчас. Данная страница служит документацией к проекту.

API

Проект предоставляет API для перевода текста. Вот основные эндпоинты:

  • POST /api/translate - Перевод текста с одного языка на другой.
  • GET /api/vocabulary - Получение словаря.
  • GET /api/translate/{id} - Получение пары слов из словаря по id.
  • GET /api/translate/requests{requestId} - Получение пар слов из словаря по id запроса.
  • POST /api/vocabulary - Добавление пары слов в словарь.
  • PUT /api/vocabulary/{id} - Обновление пары слов по id.
  • DELETE /api/vocabulary/{id} - Удаление пары слов из словаря по id.
  • DELETE /api/vocabulary - Удаление всего словаря.
