Информация о системе

Разрешённый формат данных:

При отправке данных с использованием параметров Title, Author или Content разрешается отправлять любой текст.

При отправке данных с использованием параметров Published At, Published Before и Published After разрешается в текстовое поле записывать только дату в формате “YYYY-MM-DD”.

При выборе логического оператора в следующих случаях в форме должен быть выбран либо NOT, либо EMPTY:

  • Запрос состоит только из одной единственной формы.
  • Если форм несколько, но данная форма является последней.
