Instruction
The "Upload file" button — downloads the selected file from the computer. The file should have an extension .pdf
The "Watch dictionary" buttons — show the user words that have been processed by the StandfordNLP library
StandfordNLP is a set of tools and libraries for natural language processing (NLP) provided by Stanford University. The text enters the pipeline and is subjected to various operations. In the program, the user will send an POST request, in which he will transfer the file from the computer to the system. The system will split into sentences, tokens, morphological parsing, token definition and syntax word parsing. the user can also execute the POST request by creating a file and sending the source code, file name and file extension to the system. The user can also send a GET request to get a dictionary output.
